# Freshdesk Support Bot

An intelligent customer support bot built using the Freshdesk API to automate ticket management, customer interactions, and support workflows. The bot helps businesses improve response time, streamline support operations, and enhance customer experience.

## Features

* update, and manage Freshdesk tickets
* Automated customer replies and responses
* AI-powered support assistance
* Ticket status tracking and prioritization
* Multi-channel support integration
* Fetch customer and agent details
* Real-time notifications and workflow automation
* Easy API integration with Freshdesk

## Tech Stack

* Python / Node.js
* Freshdesk API
* REST API Integration
* Automation Api's
* Ollama Ai Model

## Use Cases

* Customer support automation
* Helpdesk ticket management
* Auto-reply systems
* Internal support workflow automation
* AI chatbot integration

## Installation

```bash
git clone https://github.com/yourusername/freshdesk-support-bot.git
cd freshdesk-support-bot
pip install -r requirements.txt
```

## Configuration

Add your Freshdesk API credentials:

```env
FRESHDESK_DOMAIN=yourdomain.freshdesk.com
FRESHDESK_API_KEY=your_api_key
```

## Usage

```bash
python app.py
```

## API Capabilities

* Create Tickets
* Update Ticket Status
* Assign Agents
* Fetch Conversations
* Send Automated Responses
* Customer Data Retrieval
